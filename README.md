Run the file - 
`$ python blockchain.py`

Result - `* Running on http://localhost:5000/ (Press CTRL+C to quit)`

**Options**

-Mine block
`http://localhost:5000/mine`

-Create new transaction
`http://localhost:5000/transactions/new`

-View full chain
`http://localhost:5000/chain`
